# Todozo - Todo List
App available at [todozo.herokuapp.com](https://todozo.herokuapp.com) or [todozo.babega.com](https://todozo.babega.com)

## Features
- Sign Up with Google
- Log In with Google
- Create todo
- Delete todo
- Logout
